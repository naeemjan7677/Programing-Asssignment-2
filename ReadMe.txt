Instructions:

1. Install mysql and mysql python connector if they aren’t already installed.
2. Create a new database
3. Run CreateDB.py, CreateTeacherTable.Py, createClassTable.py, createStudentGradesTable.py. 
    Make sure you put the correct host, and user password in the python code!

4. Run main.py

In main.py you can choose to be a teacher, a parent or a student, 
student and parents can see lots of different data and teachers can add grades to student


there are 5 students in the database everyone with a different id, and 4 teachers with different ids

You will be asked to enter the name of the parent/student make sure you write it the same as in the database!
